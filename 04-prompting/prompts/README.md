# ⚠️ Manual Prompts — Fallback Only

> **You probably don't need this folder.**
>
> If your AI agent can read project files (like Gemini, Claude, Cursor, etc.), it will automatically read the templates in `01-docs/` and drive the process. Just tell it your idea and it handles the rest.
>
> **This folder is only for:** Basic AI tools (like ChatGPT web) that can't read your project files. In that case, copy-paste the relevant prompt below and fill in the `[PLACEHOLDERS]` manually.

| File | When to Use (Manual Only) |
|:-----|:--------------------------|
| `01_research.md` | Before writing anything — validate your idea |
| `02_prd.md` | Generate a Product Requirements Document |
| `03_tech_spec.md` | Generate tech stack, app flow, backend docs |
| `04_design.md` | Generate a design system |
| `05_implementation.md` | During development — implement features |
| `06_testing.md` | Write tests and run verification |
| `07_deployment.md` | Deploy and monitor |
